Kapitola 4.1: Základní koncepty - Cvičení

V této části si procvičíte základní stavební kameny programování v Pythonu, jako je výstup, proměnné, vstup, komentáře, odsazení a podmíněné příkazy.

Cvičení 1: Váš první program a výstup

Procvičujeme: Funkce print(), základní výstup textu.

Zadání: Napište program, který na obrazovku vypíše jednoduchý pozdrav, například "Ahoj, Python světe!".

Zadání: Napište program, který na obrazovku vypíše jednoduchý pozdrav, například "Ahoj, Python světe!".

Očekávaný výstup:

Ahoj, Python světe!
Ukázka řešení 

print("Ahoj, Python světe!")

Cvičení 2: Práce s proměnnými a datovými typy

Procvičujeme: Definování proměnných, přiřazení hodnot, základní datové typy (řetězec, číslo), výpis proměnných pomocí print().

Zadání: Vytvořte proměnnou jmeno a přiřaďte jí své jméno jako textový řetězec. Vytvořte proměnnou vek a přiřaďte jí svůj věk jako celé číslo. Poté pomocí funkce print() vypište větu, která bude obsahovat hodnoty obou proměnných (např. "Jmenuji se [Vaše jméno] a je mi [Váš věk] let.").

Očekávaný výstup (příklad pro jméno "Petr" a věk 30):

Jmenuji se Petr a je mi 30 let.
Ukázka řešení 

jmeno = "Petr" # Změňte na své jméno
vek = 30     # Změňte na svůj věk
print(f"Jmenuji se {jmeno} a je mi {vek} let.")

Cvičení 3: Získání a převod vstupu od uživatele

Procvičujeme: Funkce input(), získání vstupu od uživatele, převod textového vstupu na číslo (int() nebo float()).

Zadání: Napište program, který se zeptá uživatele na jeho oblíbené číslo. Získaný vstup převeďte na celé číslo a uložte do proměnné. Poté vypište zprávu, která potvrdí zadané číslo (např. "Tvoje oblíbené číslo je [zadané číslo].").

Očekávaný výstup (příklad pro vstup "42"):

Tvoje oblíbené číslo je 42.
Ukázka řešení 

cislo_str = input("Zadej své oblíbené číslo: ")
cislo = int(cislo_str)
print(f"Tvoje oblíbené číslo je {cislo}.")

Cvičení 4: Komentáře a čitelnost kódu

Procvičujeme: Použití jednořádkových komentářů (#) pro vysvětlení kódu.

Zadání: Vezměte kód z Cvičení 3 a přidejte k němu komentáře, které vysvětlí, co jednotlivé řádky kódu dělají (získání vstupu, převod na číslo, výpis). Snažte se, aby komentáře byly jasné a pomohly někomu jinému pochopit váš kód.

Očekávaný výstup:

(Výstup programu bude stejný jako v Cvičení 3, ale kód bude lépe dokumentovaný.)
Ukázka řešení 

# Získání vstupu od uživatele
cislo_str = input("Zadej své oblíbené číslo: ") # Výzva pro uživatele

# Převod textového vstupu na celé číslo
cislo = int(cislo_str)

# Výpis potvrzovací zprávy s použitím proměnné
print(f"Tvoje oblíbené číslo je {cislo}.")

Cvičení 5: Procvičování odsazení

Procvičujeme: Správné odsazení bloků kódu v Pythonu.

Zadání: Následující kód je napsán bez správného odsazení. Opravte odsazení tak, aby program fungoval správně a vypisoval zprávy podle logiky podmíněných příkazů.


# Kód s chybným odsazením - OPRAVTE HO!
cislo = 15
if cislo > 10:
print("Číslo je větší než 10.")
if cislo > 20:
print("Číslo je větší než 20.")
else:
print("Číslo není větší než 20.")
print("Konec programu.")

Očekávaný výstup (po opravě odsazení):

Číslo je větší než 10.
Číslo není větší než 20.
Konec programu.
Ukázka řešení 

# Kód s opraveným odsazením
cislo = 15
if cislo > 10:
    print("Číslo je větší než 10.")
    if cislo > 20:
        print("Číslo je větší než 20.")
    else:
        print("Číslo není větší než 20.")
print("Konec programu.")

Cvičení 6: Rozhodování s if a else

Procvičujeme: Použití příkazů if a else pro jednoduché rozhodování.

Zadání: Napište program, který se zeptá uživatele na věk. Pokud je věk 18 nebo více, vypište "Jsi dospělý.". V opačném případě (pokud je věk menší než 18) vypište "Nejsi ještě dospělý.". Nezapomeňte převést vstup na číslo.

Očekávaný výstup (příklad pro vstup "20"):

Jsi dospělý.

Očekávaný výstup (příklad pro vstup "16"):

Nejsi ještě dospělý.
Ukázka řešení 

vek_str = input("Kolik ti je let? ")
vek = int(vek_str)

if vek >= 18:
    print("Jsi dospělý.")
else:
    print("Nejsi ještě dospělý.")

Cvičení 7: Více možností s elif

Procvičujeme: Použití příkazů if, elif a else pro rozhodování mezi více možnostmi.

Zadání: Napište program, který se zeptá uživatele na známku (číslo 1-5). Podle známky vypište slovní hodnocení: - 1: "Výborně" - 2: "Chvalitebně" - 3: "Dobře" - 4: "Dostatečně" - 5: "Nedostatečně" Pokud uživatel zadá jiné číslo, vypište "Neplatná známka."

Očekávaný výstup (příklad pro vstup "2"):

Chvalitebně

Očekávaný výstup (příklad pro vstup "6"):

Neplatná známka.
Ukázka řešení 

znamka_str = input("Zadej známku (1-5): ")
znamka = int(znamka_str)

if znamka == 1:
    print("Výborně")
elif znamka == 2:
    print("Chvalitebně")
elif znamka == 3:
    print("Dobře")
elif znamka == 4:
    print("Dostatečně")
elif znamka == 5:
    print("Nedostatečně")
else:
    print("Neplatná známka.")
