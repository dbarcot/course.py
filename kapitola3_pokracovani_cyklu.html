Příkaz continue

Příkaz continue se používá uvnitř cyklů (for nebo while). Když Python narazí na continue, přeskočí zbytek kódu v aktuální iteraci cyklu a okamžitě přejde k další iteraci.

Použití continue v cyklu for


# Vypíšeme všechna čísla od 0 do 4 kromě čísla 2
for i in range(5):
    if i == 2:
        print("Přeskakuji číslo 2")
        continue # Přeskočí zbytek těla cyklu pro i=2
    print(f"Aktuální číslo: {i}")

# Výstup:
# Aktuální číslo: 0
# Aktuální číslo: 1
# Přeskakuji číslo 2
# Aktuální číslo: 3
# Aktuální číslo: 4

Použití continue v cyklu while

Stejně jako u cyklu for, continue v cyklu while přeskočí zbytek aktuální iterace a vrátí se na začátek cyklu k vyhodnocení podmínky.


# Vypíšeme čísla od 1 do 5, ale přeskočíme číslo 3
i = 0
while i < 5:
    i += 1
    if i == 3:
        print("Přeskakuji číslo 3")
        continue # Přeskočí print(i) pro i=3
    print(f"Aktuální číslo: {i}")

# Výstup:
# Aktuální číslo: 1
# Aktuální číslo: 2
# Přeskakuji číslo 3
# Aktuální číslo: 4
# Aktuální číslo: 5

Příkaz continue je užitečný, když chcete v cyklu přeskočit zpracování pro určité případy, ale nechcete cyklus úplně ukončit (jako by to udělal příkaz break).
