Importování konkrétních prvků (from ... import ...)

V kapitole o importování modulů jsme si ukázali základní příkaz import nazev_modulu. Existuje však i jiný způsob, jak importovat pouze konkrétní části modulu (funkce, třídy, proměnné) do vašeho aktuálního jmenného prostoru. K tomu slouží příkaz from nazev_modulu import prvek1, prvek2, ....

Použití from ... import ...

Při použití tohoto způsobu importu můžete k importovaným prvkům přistupovat přímo, bez nutnosti uvádět název modulu před nimi.


# Importujeme konkrétní funkce pi a sqrt z modulu math
from math import pi, sqrt

print(pi)       # Přístup přímo k pi
print(sqrt(16)) # Přístup přímo k sqrt (vypíše 4.0)

# Importujeme funkci randint z modulu random
from random import randint

nahodne_cislo = randint(1, 100) # Přístup přímo k randint
print(f"Náhodné číslo mezi 1 a 100: {nahodne_cislo}")

# Můžete importovat i více prvků najednou
from time import localtime, strftime

t = localtime() # Přístup přímo k localtime
cas_formatovany = strftime("%H:%M:%S", t) # Přístup přímo k strftime
print(f"Aktuální čas: {cas_formatovany}")

Import s přejmenováním pomocí from ... import ... as alias

Stejně jako u základního importu, můžete importované prvky přejmenovat pomocí klíčového slova as.


# Importujeme funkci sqrt z modulu math a přejmenujeme ji na odmocnina
from math import sqrt as odmocnina

vysledek = odmocnina(81)
print(f"Odmocnina z 81 je: {vysledek}") # Vypíše: Odmocnina z 81 je: 9.0

Import všech prvků (from nazev_modulu import *)

Můžete také importovat *všechny* veřejné prvky z modulu pomocí hvězdičky (*). Tento způsob se nazývá "wildcard import".


# Importuje VŠECHNY veřejné prvky z modulu math
from math import *

print(pi)
print(sqrt(49))
print(cos(0)) # Funkce cos je také součástí modulu math

Varování: Použití from modul import * se obecně nedoporučuje pro větší projekty. Může to vést k nejasnostem, odkud importované prvky pocházejí, a může způsobit kolize názvů, pokud různé moduly definují prvky se stejným jménem. Je lepší importovat buď celý modul (import nazev_modulu) nebo konkrétní prvky, které potřebujete (from nazev_modulu import prvek).

Příkaz from ... import ... je užitečný, když potřebujete často používat jen několik konkrétních funkcí nebo proměnných z modulu a chcete se vyhnout opakovanému psaní názvu modulu.
