Cvičení 8: Opakování s cyklem for (procházení sekvencí)

Procvičujeme: Cyklus for pro iteraci přes prvky seznamu.

ovoce = ["jablko", "banán", "třešeň", "kiwi"]
# Váš kód zde

jablko
banán
třešeň
kiwi
Ukázka řešení

for nazev in ovoce:

Cvičení 10: Opakování s cyklem while

Procvičujeme: Cyklus while pro opakování kódu na základě podmínky.

Zadání: Pomocí cyklu while vypište čísla od 5 dolů do 1 (včetně). Použijte proměnnou jako počítadlo a nezapomeňte ji v cyklu měnit.


# Váš kód zde
# Použijte while

Očekávaný výstup:

5
4
3
2
1
Ukázka řešení 

pocitadlo = 5
while pocitadlo >= 1:
    print(pocitadlo)
    pocitadlo -= 1 # Zkrácený zápis pro pocitadlo = pocitadlo - 1

cisla = [10, 50, 120, 30, 150, 80]

Nalezeno číslo větší než 100, končím.

Zadání: Projděte čísla od 1 do 10 pomocí cyklu for. Pokud je číslo sudé, přeskočte jeho výpis pomocí continue. Vypište pouze lichá čísla.


# Váš kód zde
# Použijte for a continue

Očekávaný výstup:

1
3
5
7
9
Ukázka řešení 

for i in range(1, 11):
    if i % 2 == 0: # Zkontroluje, zda je číslo sudé
        continue   # Pokud je sudé, přeskočí zbytek iterace
    print(i)

Cvičení 14: Funkce s parametry a návratovou hodnotou (return)

Procvičujeme: Definování funkce s parametry a použitím příkazu return pro vrácení hodnoty.

Zadání: Vytvořte funkci s názvem spocitej_obvod_ctverce, která přijme jeden parametr (délku strany čtverce). Funkce by měla vypočítat obvod čtverce (4 * strana) a vrátit tuto hodnotu pomocí return. Poté zavolejte funkci s nějakou hodnotou strany a výsledek uložte do proměnné a vypište.


# Váš kód zde
# Definujte funkci spocitej_obvod_ctverce a použijte ji

Očekávaný výstup (příklad pro stranu 5):

Obvod čtverce se stranou 5 je 20.
Ukázka řešení 

def spocitej_obvod_ctverce(strana):
    obvod = 4 * strana
    return obvod

delka_strany = 5
vysledek_obvod = spocitej_obvod_ctverce(delka_strany)
print(f"Obvod čtverce se stranou {delka_strany} je {vysledek_obvod}.")

Cvičení 24: Kombinace cyklů a podmínek

Procvičujeme: Použití podmíněných příkazů (if, elif, else) uvnitř cyklů (for nebo while).

Zadání: Projděte čísla od 1 do 15 pomocí cyklu for. Pro každé číslo zkontrolujte: - Pokud je číslo dělitelné 3 i 5, vypište "FizzBuzz". - Jinak, pokud je číslo dělitelné 3, vypište "Fizz". - Jinak, pokud je číslo dělitelné 5, vypište "Buzz". - Jinak (pokud není dělitelné ani 3 ani 5), vypište samotné číslo.


# Váš kód zde
# Použijte for, if, elif, else

Očekávaný výstup:

1
2
Fizz
4
Buzz
Fizz
7
8
Fizz
Buzz
11
Fizz
13
14
FizzBuzz
Ukázka řešení 

for cislo in range(1, 16):
    if cislo % 3 == 0 and cislo % 5 == 0:
        print("FizzBuzz")
    elif cislo % 3 == 0:
        print("Fizz")
    elif cislo % 5 == 0:
        print("Buzz")
    else:
        print(cislo)
