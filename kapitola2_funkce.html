Definování funkcí (def, return)

Funkce jsou pojmenované bloky kódu, které provádějí určitý úkol. Můžete je "zavolat" (spustit) z jiných částí programu, kolikrát potřebujete. Používání funkcí má několik výhod:

  • Opakované použití kódu: Nemusíte psát stejný kód vícekrát. Stačí ho napsat jednou do funkce a pak funkci volat.
  • Organizace kódu: Rozdělují program na menší, logické části, což zlepšuje čitelnost a usnadňuje údržbu.
  • Abstrakce: Skrývají detaily implementace. Stačí vědět, co funkce dělá, ne nutně jak to dělá.

Definice funkce pomocí def

Definice funkce začíná klíčovým slovem def, následovaným názvem funkce, závorkami () (které mohou obsahovat parametry) a dvojtečkou :. Tělo funkce (kód, který funkce provádí) musí být odsazeno.


# Definice jednoduché funkce bez parametrů a bez návratové hodnoty
def pozdrav():
    print("Ahoj! Vítej v programu.")

# Volání (spuštění) funkce
pozdrav()  # Vypíše: Ahoj! Vítej v programu.
pozdrav()  # Můžeme ji volat opakovaně

Parametry funkce

Funkce mohou přijímat vstupní hodnoty, tzv. parametry (nebo argumenty). Ty se definují v závorkách v definici funkce.


def pozdrav_jmenem(jmeno): # 'jmeno' je parametr
    print(f"Ahoj, {jmeno}!")

pozdrav_jmenem("Petře")  # "Petře" je argument předaný parametru 'jmeno'. Vypíše: Ahoj, Petře!
pozdrav_jmenem("Lucie")  # Vypíše: Ahoj, Lucie!

def secti_cisla(a, b): # Funkce se dvěma parametry
    soucet = a + b
    print(f"Součet {a} + {b} je {soucet}.")

secti_cisla(5, 3)    # Vypíše: Součet 5 + 3 je 8.
secti_cisla(100, 200) # Vypíše: Součet 100 + 200 je 300.

Návratová hodnota pomocí return

Funkce mohou nejen provádět akce (jako je tisk), ale také vracet výsledek své činnosti zpět do místa, odkud byly volány. K tomu slouží příkaz return.


def vynasob_dvema(cislo):
    vysledek = cislo * 2
    return vysledek # Funkce vrací hodnotu proměnné 'vysledek'

moje_cislo = 7
vysledek_funkce = vynasob_dvema(moje_cislo) # Výsledek volání funkce se uloží do proměnné
print(f"Dvojnásobek čísla {moje_cislo} je {vysledek_funkce}.") # Vypíše: Dvojnásobek čísla 7 je 14.

print(f"Trojnásobek je {vynasob_dvema(3) * 1.5}") # Můžeme použít výsledek přímo

# Příklad z examples/CV8_prevod_stupnu_na_radiany_a_radianu_na_stupne_Barcot_1-33.py
import math # Potřebujeme pro math.pi

def stupne_na_radiany(stupne):
    # Tato funkce převede stupně na radiány a vrátí výsledek
    return stupne * math.pi / 180

def radiany_na_stupne(radiany):
    # Tato funkce převede radiány na stupně a vrátí výsledek
    return radiany * 180 / math.pi

rad = stupne_na_radiany(90)
print(f"90 stupňů je {rad:.4f} radiánů.") # .4f formátuje na 4 desetinná místa

stup = radiany_na_stupne(math.pi / 2)
print(f"{math.pi / 2:.4f} radiánů je {stup}°.")

Pokud funkce neobsahuje příkaz return, nebo obsahuje return bez hodnoty, automaticky vrací speciální hodnotu None.

Docstringy (Dokumentační řetězce)

Je dobrým zvykem přidávat na začátek těla funkce víceřádkový řetězec (uzavřený v """..."""), který popisuje, co funkce dělá, jaké má parametry a co vrací. Tomuto se říká docstring.


def vypocitej_obsah_obdelnika(delka, sirka):
    """Vypočítá obsah obdélníka.

    Args:
        delka (float): Délka obdélníka.
        sirka (float): Šířka obdélníka.

    Returns:
        float: Obsah obdélníka.
    """
    return delka * sirka

obsah = vypocitej_obsah_obdelnika(10, 5)
print(f"Obsah obdélníka je: {obsah}")
# Nástroje a některá IDE mohou tento docstring zobrazit jako nápovědu k funkci.
