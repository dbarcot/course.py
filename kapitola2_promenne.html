Proměnné a přiřazení

Proměnné si můžeme představit jako pojmenované "krabičky" v paměti počítače, do kterých ukládáme data (například čísla, text, seznamy). Hodnotu proměnné přiřadíme pomocí operátoru rovná se (=).


# Vytvoření proměnné 'pozdrav' a uložení textu (řetězce)
pozdrav = "Ahoj světe!"
print(pozdrav)  # Vypíše: Ahoj světe!

# Vytvoření proměnné 'cislo' a uložení celého čísla
cislo = 123
print(cislo)  # Vypíše: 123

# Vytvoření proměnné 'desetinne_cislo' a uložení desetinného čísla
desetinne_cislo = 3.14
print(desetinne_cislo)  # Vypíše: 3.14

# Proměnná může změnit svou hodnotu
vek = 18
print("Původní věk:", vek)  # Vypíše: Původní věk: 18
vek = 19  # Nová hodnota pro proměnnou 'vek'
print("Nový věk:", vek)  # Vypíše: Nový věk: 19

# Můžeme také provádět operace s proměnnými
a = 10
b = 5
soucet = a + b
print("Součet a + b je:", soucet)  # Vypíše: Součet a + b je: 15

Pravidla pro názvy proměnných:

  • Název proměnné může obsahovat písmena (malá i velká), číslice a podtržítko (_).
  • Název proměnné nesmí začínat číslicí.
  • Názvy proměnných jsou citlivé na velikost písmen (vek a Vek jsou dvě různé proměnné).
  • Název proměnné by neměl být stejný jako klíčová slova Pythonu (např. if, for, def, class atd.).
  • Je dobrým zvykem používat výstižné názvy, které popisují, co proměnná uchovává (např. jmeno_uzivatele, pocet_pokusu). Pro víceslovné názvy se často používá podtržítková notace (snake_case).

# Správné názvy proměnných
moje_promenna = "hodnota"
cislo1 = 1
uzivatelsky_vstup = "text"
_pomocna_promenna = True # Začínající podtržítkem je konvence pro interní proměnné
