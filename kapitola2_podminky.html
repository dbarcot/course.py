Podmíněné příkazy (if, elif, else)

Podmíněné příkazy umožňují programu provádět různý kód v závislosti na tom, zda je určitá podmínka pravdivá (True) nebo nepravdivá (False). Jsou základem pro rozhodování v programech.

Příkaz if

Blok kódu pod příkazem if se provede pouze tehdy, pokud je podmínka za if vyhodnocena jako pravdivá (True).


vek = 20
if vek >= 18:
    print("Jste dospělý.") # Tento kód se provede, protože 20 >= 18 je True

teplota = 5
if teplota < 10:
    print("Je zima, obleč si bundu!") # Tento kód se provede

Příkaz else

Příkaz else lze použít v kombinaci s if. Blok kódu pod else se provede tehdy, pokud podmínka za if byla nepravdivá (False).


vek = 15
if vek >= 18:
    print("Jste dospělý.")
else:
    print("Nejste ještě dospělý.") # Tento kód se provede, protože 15 >= 18 je False

heslo_spravne = False
if heslo_spravne:
    print("Přístup povolen.")
else:
    print("Přístup zamítnut! Špatné heslo.") # Tento kód se provede

Příkaz elif (else if)

Příkaz elif umožňuje testovat více podmínek postupně. Pokud je podmínka za if nepravdivá, zkontroluje se podmínka prvního elif. Pokud i ta je nepravdivá, zkontroluje se další elif, a tak dále. Pokud žádná z podmínek if nebo elif není pravdivá, provede se blok else (pokud je přítomen).


# Příklad z examples/CV2_ovoce_a_zelenina_Barcot_1-33.py
ovoce = ["jablko", "hruska", "merunka", "tresen"] # V původním souboru jsou i další
zelenina = ["mrkev", "celer", "cibule", "rajce"] # V původním souboru jsou i další
x = input("Zadejte ovoce či zeleninu: ")

if x in ovoce:
    print("Zadal jste ovoce.")
elif x in zelenina:
    print("Zadal jste zeleninu.")
else:
    print("Toto slovo neznám.")

# Další příklad s čísly
skore = 75
if skore >= 90:
    print("Známka: Výborně (A)")
elif skore >= 70:
    print("Známka: Chvalitebně (B)") # Tento kód se provede (75 >= 70)
elif skore >= 50:
    print("Známka: Dobře (C)")
else:
    print("Známka: Nedostatečně (F)")

Podmínky mohou být složitější a používat logické operátory jako and (a zároveň), or (nebo), not (negace).


vek = 25
ma_ridicsky_prukaz = True

if vek >= 18 and ma_ridicsky_prukaz:
    print("Můžete řídit auto.")
else:
    print("Nemůžete řídit auto.")
