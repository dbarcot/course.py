Cyklus for

Cyklus for v Pythonu se používá k iteraci (procházení) přes sekvenci (jako je seznam, n-tice, slovník, množina nebo řetězec) nebo jiné iterovatelné objekty. S každou iterací cyklus přiřadí aktuální prvek sekvence do proměnné a provede blok kódu.

Procházení seznamu


ovoce_seznam = ["jablko", "banán", "třešeň"]
for o in ovoce_seznam:  # Proměnná 'o' postupně nabývá hodnot "jablko", "banán", "třešeň"
    print(f"Mám rád/a {o}.")
# Výstup:
# Mám rád/a jablko.
# Mám rád/a banán.
# Mám rád/a třešeň.

Procházení řetězce

Řetězec je sekvence znaků, takže přes něj můžeme také iterovat.


slovo = "Python"
for pismeno in slovo:
    print(pismeno)
# Výstup:
# P
# y
# t
# h
# o
# n

Funkce range()

Velmi často chceme provést blok kódu určitý početkrát. K tomu slouží funkce range(), která generuje sekvenci čísel.

  • range(stop): Generuje čísla od 0 do stop-1.
  • range(start, stop): Generuje čísla od start do stop-1.
  • range(start, stop, step): Generuje čísla od start do stop-1 s krokem step.

# range(stop)
for i in range(5):  # i bude 0, 1, 2, 3, 4
    print(f"Iterace číslo: {i}")

# range(start, stop)
for i in range(2, 6): # i bude 2, 3, 4, 5
    print(f"Číslo: {i}")

# range(start, stop, step)
for i in range(1, 10, 2): # i bude 1, 3, 5, 7, 9
    print(f"Liché číslo: {i}")

Vnořené cykly

Můžeme mít cyklus uvnitř jiného cyklu. To se hodí například pro práci s dvourozměrnými strukturami (jako jsou tabulky nebo matice).


for i in range(1, 4): # Vnější cyklus (1, 2, 3)
    for j in ["a", "b"]: # Vnitřní cyklus ("a", "b")
        print(f"i={i}, j={j}")
# Výstup:
# i=1, j=a
# i=1, j=b
# i=2, j=a
# i=2, j=b
# i=3, j=a
# i=3, j=b

Cyklus for je velmi mocný nástroj pro opakování operací nad kolekcemi dat nebo pro provádění kódu daný početkrát.
