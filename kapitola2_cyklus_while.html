Cyklus while

Cyklus while (dokud) opakuje blok kódu, dokud je zadaná podmínka pravdivá (True). Před každým provedením těla cyklu se podmínka znovu vyhodnotí.

Základní struktura


# Jednoduchý příklad while cyklu
pocitadlo = 0
while pocitadlo < 5:  # Podmínka: dokud je pocitadlo menší než 5
    print(f"Pocitadlo je: {pocitadlo}")
    pocitadlo = pocitadlo + 1 # Důležité: měníme hodnotu proměnné v podmínce!
                            # Bez toho by cyklus běžel donekonečna.
print("Konec while cyklu.")
# Výstup:
# Pocitadlo je: 0
# Pocitadlo je: 1
# Pocitadlo je: 2
# Pocitadlo je: 3
# Pocitadlo je: 4
# Konec while cyklu.

Pozor na nekonečné cykly! Pokud se podmínka cyklu while nikdy nestane nepravdivou (False), cyklus poběží donekonečna a program se "zasekne". Vždy se ujistěte, že se v těle cyklu děje něco, co může podmínku ovlivnit.

Použití s příkazem break

Někdy chceme vytvořit cyklus, který běží "donekonečna", dokud nenastane určitá událost. V takovém případě můžeme použít while True: a uvnitř cyklu testovat podmínku pro ukončení pomocí příkazu break.

# Jednodušší příklad s break
cislo = 0
while True:
    print(f"Aktuální číslo: {cislo}")
    if cislo == 3:
        print("Dosáhli jsme čísla 3, končíme.")
        break # Ukončí cyklus
    cislo += 1 # Zkrácený zápis pro cislo = cislo + 1

Použití s příkazem continue (zmíníme stručně)

Příkaz continue přeskočí zbytek aktuální iterace cyklu a pokračuje další iterací (znovu vyhodnotí podmínku).


i = 0
while i < 5:
    i += 1
    if i == 3:
        print("Přeskakuji číslo 3")
        continue # Přeskočí print(i) pro i=3
    print(i)
# Výstup:
# 1
# 2
# Přeskakuji číslo 3
# 4
# 5

Cyklus while je vhodný, když nevíme předem, kolikrát se má cyklus opakovat, a opakování závisí na splnění nějaké dynamické podmínky.
