Importování modulů (import)

Python má rozsáhlou standardní knihovnu modulů, které poskytují mnoho užitečných funkcí a nástrojů. Modul je v podstatě soubor s Python kódem (.py), který obsahuje definice funkcí, tříd a proměnných. Abychom mohli tyto prvky z modulu použít ve svém programu, musíme modul nejprve "importovat".

Základní import pomocí import nazev_modulu

Tento způsob importuje celý modul. K prvkům definovaným v modulu pak přistupujeme pomocí tečkové notace: nazev_modulu.nazev_prvku.


# Příklad s modulem 'math', který obsahuje matematické funkce a konstanty
import math

print(math.pi)  # Vypíše hodnotu konstanty Pí (cca 3.14159...)
print(math.sqrt(16)) # Vypíše odmocninu z 16, tj. 4.0

# Příklad z examples/CV6_aktualni_hodina_+_pozdrav_Barcot_1-33.py
import time # Importuje celý modul 'time' pro práci s časem

t = time.localtime() # Funkce z modulu time
hodina = t.tm_hour   # Přístup k atributu objektu vráceného funkcí localtime()
print(f"Aktuální hodina: {hodina}")

# Příklad z examples/CV11_JSON_format_s_nahodnymi_cisly_Barcot_1-33.py
import random # Modul pro generování náhodných čísel
import json   # Modul pro práci s datovým formátem JSON

nahodne_cislo = random.randint(1, 100) # Funkce z modulu random
print(f"Náhodné číslo mezi 1 a 100: {nahodne_cislo}")

data_pro_json = {"jmeno": "Karel", "vek": random.randint(20, 60)}
json_string = json.dumps(data_pro_json, indent=4) # Funkce z modulu json
print("Data ve formátu JSON:")
print(json_string)

Import s přejmenováním pomocí import nazev_modulu as alias

Někdy je název modulu příliš dlouhý nebo koliduje s jiným názvem ve vašem kódu. Můžete ho importovat pod jiným, kratším jménem (aliasem).


import math as m # Modul math bude nyní dostupný pod aliasem 'm'

print(m.pi)
print(m.pow(2, 3)) # Výpočet 2 na 3, tj. 8.0

import datetime as dt # Častý alias pro modul datetime

dnes = dt.date.today()
print(f"Dnešní datum: {dnes}")

Import konkrétních prvků pomocí from nazev_modulu import prvek1, prvek2

Pokud potřebujete z modulu jen několik konkrétních funkcí nebo proměnných, můžete je importovat přímo do aktuálního jmenného prostoru. Pak k nim přistupujete bez nutnosti psát název modulu.


from math import pi, sqrt, pow

print(pi)       # Přístup přímo k pi
print(sqrt(25)) # Přístup přímo k sqrt
print(pow(3, 2))  # Přístup přímo k pow

# Můžete také importovat vše z modulu (obecně se nedoporučuje kvůli možné kolizi názvů)
# from random import *
# print(randint(1, 10)) # randint je nyní přímo dostupná

Varování: Použití from modul import * (tzv. "wildcard import") se obecně nedoporučuje pro větší projekty, protože může vést k nejasnostem ohledně toho, odkud pochází určitá funkce nebo proměnná, a může způsobit kolize názvů, pokud různé moduly definují prvky se stejným jménem.

Kde Python hledá moduly?

Python hledá moduly v několika místech v určitém pořadí:

  1. V aktuálním adresáři, kde je spuštěn skript.
  2. V adresářích uvedených v proměnné prostředí PYTHONPATH (pokud je nastavena).
  3. Ve standardních instalačních adresářích Pythonu (kde je uložena standardní knihovna).

Můžete také vytvářet vlastní moduly (jednoduše soubory .py s vaším kódem) a importovat je do jiných svých skriptů, pokud jsou ve správném umístění.
