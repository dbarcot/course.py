Odsazení v Pythonu

Odsazení (počet mezer nebo tabulátorů na začátku řádku) je v Pythonu extrémně důležité! Na rozdíl od mnoha jiných programovacích jazyků, kde se bloky kódu (například co patří k podmínce if nebo cyklu for) vyznačují složenými závorkami { } nebo klíčovými slovy jako begin/end, v Pythonu se k tomuto účelu používá právě odsazení.

Všechny řádky kódu, které patří do stejného bloku, musí být odsazeny o stejný počet mezer. Standardní a doporučený způsob je používat 4 mezery pro každou úroveň odsazení.

Proč je odsazení důležité?

Python interpret používá odsazení k určení, které příkazy jsou součástí kterého bloku. Pokud odsazení není konzistentní nebo je chybné, program buď nebude fungovat správně, nebo Python vyvolá chybu IndentationError.


# Příklad správného odsazení
cislo = 7
if cislo > 5:
    print("Číslo je větší než 5.")  # Tento řádek je odsazený, patří k bloku if
    print("Toto je také součást bloku if.") # Stejné odsazení, také patří k if
    if cislo < 10:
        print("A zároveň je menší než 10.") # Další úroveň odsazení pro vnořený if
print("Tento řádek se vykoná vždy, nepatří k žádnému bloku if výše.") # Není odsazený

Většina moderních textových editorů a vývojových prostředí (IDE) vám pomůže se správným odsazováním, například automatickým vložením 4 mezer po stisku klávesy Tab. Je dobré si nastavit editor tak, aby klávesa Tab vkládala mezery místo skutečného znaku tabulátoru, aby se předešlo problémům s kombinováním mezer a tabulátorů (což Python také nemá rád).

Správné a konzistentní odsazení nejenže zajišťuje funkčnost kódu v Pythonu, ale také výrazně přispívá k jeho čitelnosti.
