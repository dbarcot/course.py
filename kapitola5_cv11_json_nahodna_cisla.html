Příklad: JSON formát s náhodnými čísly (CV11_JSON_format_s_nahodnymi_cisly_Barcot_1-33.py)

Tento program ukazuje, jak generovat náhodná data pomocí modulu random, ukládat je do Python slovníků a seznamů a poté tato data serializovat do formátu JSON a uložit do souboru pomocí modulu json.


import random # Import modulu pro generování náhodných čísel
import json   # Import modulu pro práci s formátem JSON

# Vytvoření prázdného seznamu pro ukládání jednotlivých měření
mereni = []

# Cyklus pro vytvoření 10 záznamů měření
for i in range(1, 11):  # Opakuje se 10krát (pro i od 1 do 10)
    # Generování seznamu 10 náhodných celých čísel mezi 0 a 100
    # Používá se list comprehension pro stručnější zápis
    cisla = [random.randint(0, 100) for _ in range(10)]
    # Vytvoření slovníku pro aktuální měření
    mereni.append({
        "poradi": i,      # Klíč "poradi" s hodnotou aktuálního čísla iterace
        "cisla": cisla    # Klíč "cisla" s hodnotou seznamu náhodných čísel
    })

# Zabalení seznamu měření do hlavního slovníku s klíčem "mereni"
vystup = {"mereni": mereni}

# Uložení výsledného slovníku do JSON souboru
# Otevření souboru "mereni.json" v režimu zápisu ('w')
# Konstrukce 'with' zajistí automatické zavření souboru
with open("mereni.json", "w", encoding="utf-8") as soubor: # Přidáno encoding pro správné kódování
    # Serializace Python objektu (slovníku 'vystup') do JSON formátu a zápis do souboru
    # indent=4 zajistí formátovaný výstup s odsazením 4 mezer pro lepší čitelnost
    json.dump(vystup, soubor, indent=4, ensure_ascii=False) # Přidáno ensure_ascii=False pro správné zobrazení české diakritiky, pokud by byla v datech

# Program po dokončení zápisu automaticky zavře soubor díky 'with' a ukončí se.
# Na obrazovku se nic nevypisuje, pokud nedojde k chybě.
