Příkaz break

Příkaz break se používá k okamžitému ukončení nejbližšího vnitřního cyklu (for nebo while), ve kterém se nachází. Jakmile Python narazí na break, provádění cyklu se zastaví a program pokračuje prvním příkazem za cyklem.

Použití break v cyklu for


# Hledání prvního čísla dělitelného 7 v seznamu
cisla = [1, 5, 12, 21, 8, 14, 30]
hledane_cislo = None # Proměnná pro uložení nalezeného čísla

for cislo in cisla:
    print(f"Kontroluji číslo: {cislo}")
    if cislo % 7 == 0: # Zbytek po dělení 7 je 0?
        hledane_cislo = cislo
        print(f"Nalezeno první číslo dělitelné 7: {hledane_cislo}")
        break # Našli jsme, co jsme hledali, cyklus může skončit
    # Pokud bychom zde neměli break, cyklus by pokračoval a našel by i 14.

if hledane_cislo is not None:
    print(f"První nalezené číslo dělitelné 7 je {hledane_cislo}.")
else:
    print("V seznamu nebylo žádné číslo dělitelné 7.")

# Výstup:
# Kontroluji číslo: 1
# Kontroluji číslo: 5
# Kontroluji číslo: 12
# Kontroluji číslo: 21
# Nalezeno první číslo dělitelné 7: 21
# První nalezené číslo dělitelné 7 je 21.

Použití break v cyklu while

Příkaz break je velmi častý v cyklech while True:, kde slouží jako hlavní mechanismus pro ukončení cyklu na základě nějaké podmínky testované uvnitř těla cyklu.


# Příklad z examples/CV2_ovoce_a_zelenina_Barcot_1-33.py
# while True:
#     pokracovat = input ("Chcete zadat nové slovo (a/n)? ")
#     if pokracovat == "a":
#         # ... (kód pro zpracování vstupu) ...
#         pass # pass je zástupný symbol, nedělá nic
#     else: # Předpokládáme, že pokud to není "a", chceme skončit
#         print("Děkujeme za použití programu")
#         break # Ukončí while True cyklus

# Jiný příklad: Hádání čísla
tajne_cislo = 7
pokus = 0
while True:
    pokus += 1
    zadane_cislo_text = input(f"Pokus č. {pokus}: Hádej tajné číslo (1-10): ")
    zadane_cislo = int(zadane_cislo_text)

    if zadane_cislo == tajne_cislo:
        print(f"Gratuluji! Uhodl jsi číslo {tajne_cislo} na {pokus}. pokus.")
        break # Správný tip, konec cyklu
    elif zadane_cislo < tajne_cislo:
        print("Tajné číslo je větší.")
    else:
        print("Tajné číslo je menší.")

    if pokus >= 3: # Omezení počtu pokusů
        print(f"Bohužel, neuhodl jsi. Tajné číslo bylo {tajne_cislo}.")
        break # Konec cyklu po 3 neúspěšných pokusech

Příkaz break je užitečný nástroj pro řízení toku cyklů, když potřebujeme cyklus opustit dříve, než by normálně skončil (tj. než by jeho podmínka byla nepravdivá nebo než by došly prvky v sekvenci).
