Kapitola 1: Úvod

Vítejte v Python tutoriálu! Tento kurz je navržen tak, aby vás provedl základy programování v Pythonu krok za krokem.

Začneme úplnými základy, takže se nemusíte bát, pokud s programováním teprve začínáte. Postupně se dostaneme k složitějším konceptům.

Python je populární, všestranný a snadno čitelný programovací jazyk. Používá se pro vývoj webových aplikací, analýzu dat, umělou inteligenci, vědecké výpočty a mnoho dalšího.

Váš první program

Tradicí v programování je začít programem, který vypíše "Ahoj světe!". V Pythonu je to velmi jednoduché:


# Toto je ukázkový Python kód
# Váš první program může vypadat takto:
print("Ahoj světe!")
