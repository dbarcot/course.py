Cvičení 31: Jednoduchý převodník jednotek (Stupně/Radiány)

Procvičujeme: Funkce, vstup/výstup, podmínky, moduly (math).

Zadání: Napište program, který funguje jako jednoduchý převodník mezi stupni a radiány. Program by měl:

  1. Zobrazit uživateli menu s možnostmi: "1 - Stupně na radiány", "2 - Radiány na stupně", "3 - Konec".
  2. Načíst volbu uživatele.
  3. Pokud uživatel zvolí 1, zeptat se na hodnotu ve stupních, provést převod na radiány a výsledek vypsat.
  4. Pokud uživatel zvolí 2, zeptat se na hodnotu v radiánech, provést převod na stupně a výsledek vypsat.
  5. Pokud uživatel zvolí 3, program ukončit.
  6. Pokud uživatel zadá neplatnou volbu, vypište chybovou zprávu a znovu zobrazte menu.
  7. Použijte funkce pro převody a pro zobrazení menu/načtení volby.

# Váš kód zde
# Použijte funkce, input(), if/elif/else, while True, break, modul math

Očekávaný výstup (příklad interakce):

Vyberte možnost:
1 - Stupně na radiány
2 - Radiány na stupně
3 - Konec
Zadejte volbu: 1
Zadejte hodnotu ve stupních: 90
90.0 stupňů je přibližně 1.5708 radiánů.

Vyberte možnost:
1 - Stupně na radiány
2 - Radiány na stupně
3 - Konec
Zadejte volbu: 2
Zadejte hodnotu v radiánech: 3.14159
3.14159 radiánů je přibližně 179.9996 stupňů.

Vyberte možnost:
1 - Stupně na radiány
2 - Radiány na stupně
3 - Konec
Zadejte volbu: 4
Neplatná volba.

Vyberte možnost:
1 - Stupně na radiány
2 - Radiány na stupně
3 - Konec
Zadejte volbu: 3
Konec programu.
Ukázka řešení 

import math

def stupne_na_radiany(stupne):
    """Převádí stupně na radiány."""
    return stupne * (math.pi / 180)

def radiany_na_stupne(radiany):
    """Převádí radiány na stupně."""
    return radiány * (180 / math.pi)

def vypis_menu():
    """Vypíše menu možností."""
    print("\nVyberte možnost:")
    print("1 - Stupně na radiány")
    print("2 - Radiány na stupně")
    print("3 - Konec")

def nacti_volbu():
    """Načte a vrátí volbu uživatele."""
    volba_str = input("Zadejte volbu: ")
    return volba_str

def proved_prevod(volba):
    """Provede převod na základě volby."""
    try:
        if volba == '1':
            hodnota_str = input("Zadejte hodnotu ve stupních: ")
            stupne = float(hodnota_str)
            radiany = stupne_na_radiany(stupne)
            print(f"{stupne} stupňů je přibližně {radiany:.4f} radiánů.")
        elif volba == '2':
            hodnota_str = input("Zadejte hodnotu v radiánech: ")
            radiany = float(hodnota_str)
            stupne = radiany_na_stupne(radiany)
            print(f"{radiany} radiánů je přibližně {stupne:.4f} stupňů.")
        else:
            print("Neplatná volba.")
    except ValueError:
        print("Chyba: Zadal jsi neplatné číslo.")
    except Exception as e:
        print(f"Nastala neočekávaná chyba: {e}")


# Hlavní část programu
while True:
    vypis_menu()
    volba = nacti_volbu()

    if volba == '3':
        print("Konec programu.")
        break # Ukončí hlavní cyklus
    else:
        proved_prevod(volba)

Cvičení 34: Implementace jednoduchého textového analyzátoru (Hledání nejdelších slov)

Procvičujeme: Práce se soubory (čtení), řetězce (split), seznamy, cykly (for), podmínky, hledání maxima.

Zadání: Napište program, který načte text ze souboru (např. text_analyza.txt), najde v něm nejdelší slovo (nebo slova, pokud jich je více se stejnou maximální délkou) a vypíše je spolu s jejich délkou. Program by měl ošetřit případ, kdy soubor neexistuje.


# Váš kód zde
# Vytvořte soubor text_analyza.txt ručně s nějakým textem
# Použijte try/except, with open(), read(), split(), for, if, len(), seznamy

Očekávaný výstup (příklad pro soubor s textem "Toto je krátký text pro testování nejdelšího slova."):

Nejdelší slovo/slova (délka 11): ['testování']

Očekávaný výstup (příklad pro soubor s textem "Ahoj světe, programování je zábava."):

Nejdelší slovo/slova (délka 13): ['programování']
Ukázka řešení 

# Před spuštěním si vytvořte soubor 'text_analyza.txt' s textem.

def najdi_nejdelsi_slova(soubor_cesta):
    """Najde nejdelší slovo/slova v textovém souboru."""
    nejdelsi_slova = []
    max_delka = 0

    try:
        with open(soubor_cesta, 'r', encoding='utf-8') as f:
            text = f.read()
            # Rozdělíme text na slova (jednoduché dělení podle mezer a interpunkce)
            # Pro robustnější řešení by bylo potřeba složitější zpracování textu
            slova = text.replace('.', '').replace(',', '').replace('!', '').replace('?', '').split()

            if not slova: # Pokud je soubor prázdný nebo obsahuje jen prázdné znaky
                return 0, []

            # Najdeme maximální délku slova
            for slovo in slova:
                if len(slovo) > max_delka:
                    max_delka = len(slovo)

            # Najdeme všechna slova s maximální délkou
            for slovo in slova:
                if len(slovo) == max_delka:
                    nejdelsi_slova.append(slovo)

            # Odstraníme duplicity a vrátíme seznam
            return max_delka, list(set(nejdelsi_slova))

    except FileNotFoundError:
        print(f"Chyba: Soubor '{soubor_cesta}' nebyl nalezen.")
        return 0, []
    except Exception as e:
        print(f"Nastala chyba při čtení souboru: {e}")
        return 0, []

# Hlavní část programu
soubor_s_textem = "text_analyza.txt"
delka, slova_vysledek = najdi_nejdelsi_slova(soubor_s_textem)

if slova_vysledek:
    print(f"Nejdelší slovo/slova (délka {delka}): {slova_vysledek}")
else:
    print("V souboru nebyla nalezena žádná slova nebo nastala chyba.")

Cvičení 35: Generátor náhodných dat a ukládání do JSON

Procvičujeme: Moduly (random, json), slovníky, seznamy, cykly (for), práce se soubory (zápis).

Zadání: Napište program, který vygeneruje seznam náhodných dat o fiktivních uživatelích a uloží je do souboru ve formátu JSON. Program by měl:

  • Vygenerovat seznam 5 uživatelů.
  • Každý uživatel by měl být slovník s klíči "id" (náhodné číslo 1000-9999), "jmeno" (náhodně vybrané ze seznamu jmen), "vek" (náhodné číslo 18-60).
  • Použijte moduly random a json.
  • Uložte seznam uživatelů do souboru uzivatele.json ve formátovaném JSON tvaru.

import random
import json

jmena = ["Alice", "Bob", "Charlie", "David", "Eva", "Filip", "Gabriela"]
# Váš kód zde
# Vygenerujte data a uložte do JSON souboru

Očekávaný výstup:

(Program nevypíše nic na obrazovku, ale vytvoří soubor 'uzivatele.json' s obsahem podobným tomuto - hodnoty se budou lišit)
[
    {
        "id": 5678,
        "jmeno": "Alice",
        "vek": 35
    },
    {
        "id": 1234,
        "jmeno": "David",
        "vek": 28
    },
    ... (další 3 uživatelé)

Ukázka řešení 

import random
import json

jmena = ["Alice", "Bob", "Charlie", "David", "Eva", "Filip", "Gabriela"]
uzivatele = []

for i in range(5): # Vygenerujeme 5 uživatelů
    uzivatel = {
        "id": random.randint(1000, 9999),
        "jmeno": random.choice(jmena),
        "vek": random.randint(18, 60)
    }
    uzivatele.append(uzivatel)

# Převod seznamu slovníků na JSON řetězec s odsazením
json_string_uzivatele = json.dumps(uzivatele, indent=4, ensure_ascii=False)

# Zápis JSON řetězce do souboru
try:
    with open("uzivatele.json", "w", encoding="utf-8") as soubor:
        soubor.write(json_string_uzivatele)
    print("Soubor 'uzivatele.json' byl úspěšně vytvořen s daty uživatelů.")
except Exception as e:
    print(f"Nastala chyba při zápisu JSON do souboru: {e}")

Cvičení 36: Jednoduché odpočítávání

Procvičujeme: Vstup od uživatele, cyklus (while nebo for), modul (time - volitelně pro pauzu), výstup.

Zadání: Napište program, který se zeptá uživatele na počet sekund pro odpočítávání. Program by měl poté odpočítávat od zadaného čísla dolů k nule, vypisovat každé číslo na nový řádek. Po dosažení nuly vypište zprávu "Start!". Můžete volitelně použít time.sleep(1) z modulu time pro pauzu jedné sekundy mezi čísly.


# Váš kód zde
# Získejte vstup, použijte cyklus, print()
# Volitelně importujte time a použijte time.sleep(1)

Očekávaný výstup (příklad pro vstup "3"):

3
2
1
Start!
Ukázka řešení 

import time # Volitelné, pro pauzu

sekundy_str = input("Zadejte počet sekund pro odpočítávání: ")
try:
    sekundy = int(sekundy_str)

    if sekundy < 0:
        print("Počet sekund musí být nezáporný.")
    else:
        # Odpočítávání pomocí while cyklu
        while sekundy > 0:
            print(sekundy)
            time.sleep(1) # Pauza 1 sekunda (vyžaduje import time)
            sekundy -= 1

        print("Start!")

except ValueError:
    print("Chyba: Nezadal jsi platné celé číslo.")
